# English Academy 🇬🇧⚓

Sito didattico per l'apprendimento dell'inglese con sezione dedicata alla **Marina Militare**.

## 🌐 Sito online
Visita: `https://TUO-USERNAME.github.io/english-academy`

## 📁 Struttura
```
/
└── index.html       ← pagina principale (tutto in un file)
```

## 🚀 Come pubblicare su GitHub Pages

1. Crea un nuovo repository su GitHub (es. `english-academy`)
2. Carica `index.html` nella root del repo
3. Vai su **Settings → Pages**
4. In **Source** seleziona: `GitHub Actions`
5. Il sito sarà online automaticamente ad ogni push su `main`

## ✨ Funzionalità
- 🏠 Home, 📚 Corsi, ⚓ Marina Militare, 🎯 Quiz, 👥 Chi Siamo, ✉️ Contatti
- Quiz con **100 domande** su navi, armi ed equipaggiamenti navali
- Inserimento nome studente e punteggio finale
- Funziona **offline** (file singolo, nessun backend)
